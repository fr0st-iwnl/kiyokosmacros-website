1. How does fatigue estimate work?

Fatigue estimate helps you decide until what fatigue you want the macro to stop. It's calculated by dividing your total fatigue by the duration of the session.


Example : Imagine you want to macro till 65 fatigue. When you do one session, you get 0.05 per session.

Fatigue Estimate = 65 / 0.05 = 1300 Rounds

2. Why is my macro not working?

Possible issues:

  • If an error appears saying "The data file "Icons" folder is missing Extract file," check this Video to see how to extract the macro.
  • Download the v1.1 (deprecated) version of AutoHotkey. If you have the v2.0 version UNINSTALL IT
  • To run the macro smoothly, your screen resolution should be at least 1920x1080. If it's lower, the macro might not work properly.
  • Weight Color Detection Issue: The issue with the macro might be due to your Roblox skin color or the green lights from above at the gym. Try centering your camera and zooming in a bit.
  • You need to have your keyboard language set to English [US].
  • If you have color filters on, turn them OFF, and also check if your NVIDIA & AMD (Restore the settings to Default) color filters are normal so the macro can accurately detect pixel and image searches.
  • If you have the new Roblox UI :

    • 1. Download the Bloxstrap.bat tool

    2. Select 1. Install Bloxstrap

    3. Once Bloxstrap is installed, select 2. Modify ClientAppSettings.json [REVERT TO OLD UI] to revert to the previous user interface.

    4. Launch Roblox using your browser after completing the previous steps.

  • Turn off any apps or settings that modify the game's display, like roshade/reshade or other optimization tools, as they may interfere with the macro.
  • High ping can interfere with the macro's performance.
  • Do NOT USE any custom fonts.
  • TURN OFF your voice chat to avoid any potential issues with the macro.
